## 🛠️ Stack Tecnológico Profesional
* **Lenguajes de Consulta y Programación:** SQL (PostgreSQL, queries avanzados, subconsultas, CTEs, INTERSECT) y Python (Pandas, NumPy, SciPy, Statsmodels).
* **Visualización de Datos y BI:** Power BI (Diseño UX/UI de dashboards, modelado de datos estrella, DAX avanzado), Seaborn y Matplotlib.
* **Metodologías Analíticas:** Análisis de cohortes, optimización de funnels de conversión, pruebas A/B de proporciones (Z-test), análisis de estacionalidad comercial y framework estructural SCQA.
* **Entorno y Control de Versiones:** Git (Flujos de trabajo Gitflow a través de terminal), VS Code, entornos virtuales dedicados (`env`) y Linux (Ubuntu).
* **Productividad y Documentación:** Obsidian (Markdown bi-direccional para bases de conocimiento).

## 📈 Estructura Estándar de los Proyectos
Cada entregable dentro de este repositorio cumple rigurosamente con un ciclo de análisis profesional enfocado al negocio:
1. **Definición del Problema (Framework SCQA):** Planteamiento de la Situación, Complicación, Pregunta analítica y Respuesta sugerida.
2. **Preparación y ETL de Datos:** Limpieza de nulos, estandarización rigurosa de formatos (columnas de Fecha y normalización de Ventas Semanales).
3. **Análisis Exploratorio de Datos (EDA):** Extracción de insights cuantitativos mediante distribuciones y correlaciones estadísticas.
4. **Visualización Interactiva o Estática:** Creación de gráficos o cuadros de mando intuitivos para audiencias técnicas y ejecutivas.
5. **Conclusiones Estratégicas:** Recomendaciones accionables basadas estrictamente en la evidencia de los datos recolectados.
