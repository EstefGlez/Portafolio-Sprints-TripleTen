<h2>📈 Estructura Estándar de los Proyectos</h2>
<p>Cada entregable dentro de este repositorio cumple rigurosamente con un ciclo de análisis profesional enfocado al negocio:</p>
<ol>
  <li><strong>Definición del Problema (Framework SCQA):</strong> Planteamiento de la Situación, Complicación, Pregunta analítica y Respuesta sugerida.</li>
  <li><strong>Preparación y ETL de Datos:</strong> Limpieza de nulos, estandarización rigurosa de formatos (columnas de Fecha y normalización de Ventas Semanales).</li>
  <li><strong>Análisis Exploratorio de Datos (EDA):</strong> Extracción de insights cuantitativos mediante distribuciones y correlaciones estadísticas.</li>
  <li><strong>Visualización Interactiva o Estática:</strong> Creación de gráficos o cuadros de mando intuitivos para audiencias técnicas y ejecutivas.</li>
  <li><strong>Conclusiones Estratégicas:</strong> Recomendaciones accionables basadas estrictamente en la evidencia de los datos recolectados.</li>
</ol>
